Questions

  1. The form tag has an "action" attribute. What does it do? It tells the form where to send the data when submitted.
  2. The form tag has a "method" attribute. What does it do? It tells the form how the data should be sent to the server.
  3. What is the difference between a "POST" and a "GET"? POST is when something changes, GET is when you search or retrieve data.
  4. What are different ways to style your form? You can use CSS to style your form elements, including colors, fonts, and layout.
  5. How does flex help for both Desktop and Mobile phones? Flex makes it easier to create layouts that adapt to different screen sizes.
  6. What is JavaScript? How did it get its name? JavaScript is a programming language that is used to create interactive web pages. It was originally called LiveScript but was renamed to JavaScript.
  7. What are the two ways you can run JavaScript? You can run JavaScript in the browser or on the server using Node.js.
  8. What was the most popular front-end js library? jQuery was the most popular front-end JavaScript library for many years.
  9. What is the most popular front-end js library today? React is currently the most popular front-end JavaScript library.
  10. What is AJAX? AJAX is a technique used in web development that lets a webpage send and receive data from a server without reloading the page.
  11. Why do you put your JS Code in a separate file? Putting JavaScript code in a separate file helps keep the HTML file clean and organized, you can use the same JS file across multiple pages, and it allows the browser to cache the file for better performance.
