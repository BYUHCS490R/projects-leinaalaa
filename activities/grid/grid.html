Questions

  1. Please describe the parent-child relationship in HTML and why it becomes essential when using flex and grid. A parent element contains child elements and this is essential in Flexbox and Grid because the parent controls how its direct children are laid out.
  2. What are the two main differences between CSS Grid and CSS Flex? Grid is a two-dimensional layout system while Flexbox is a one-dimensional layout system. Grid allows more precise layout control compared to Flexbox.
