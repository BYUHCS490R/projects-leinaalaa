Questions

  1. Why is the viewport meta tag important? It controls the different scaling and layouts on different devices.
  2. How can you adjust the CSS for printing? You can use media queries for print styles.
  3. When would you want to adjust the CSS for printing? When you want the page to look clean and readable when printed.
  4. Why do we need a viewport meta tag? It ensures the page is displayed correctly on different devices.
  5. When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority? The one that appears last in the CSS file takes priority.
  6. How do you change the CSS for a web page displayed on a phone? You can use media queries to apply different styles based on the device's screen size.
