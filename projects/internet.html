ASCII and Unicode

ACSII is a a coding system for American English characters. Unicode is a system that support many languages.

What is UTF-8?

A unicode format that allows a wide range of characters from different languages.

URL

The address used to locate on the internet.

Domain Name

A name that identifies a website.

HTTP

A protocol used to transfer web pages from a browser and a server.

HTTPS

A secure version of HTTP that encrypts data.

IP Address

A string of numbers that identifies each computer.

DNS

Translates domain names into IP addresses.

W3C Recommendations

A standard or guideline made by the World Wide Wed Consortium.

XML, XHTML, HTML, HTML5

XML: A markup language for storing and transporting data. XHTML: An application of XML, improves the structure of web documents. HTML: The standard language for making web pages. HTML5: Latest version of HTML with modern features.
